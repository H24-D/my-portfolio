## Files

```
index.html   → page structure and content
style.css    → styling and animations
script.js    → navigation and typed text effect
```

---

## Run Locally

Download the files and open `index.html` in your browser. No installation needed.

For auto-reload while editing, use the **Live Server** extension in VS Code.

---

## Deploy

Push to GitHub and connect the repo to [Vercel](https://vercel.com) or [Netlify](https://netlify.com). Any new `git push` will auto-deploy.
